## 🧠 ЧТО УМЕЕТ JARVIS СЕЙЧАС

### Layer 0: **VerbEngine** (80+ действий)
- Понимает русские глаголы: "найди", "открой", "удали", "посчитай"...
- Гибкое сопоставление синонимов

### Layer 1: **Parser + NLP_minimal**
- Токенизация русских текстов
- Нормализация слов (стемминг)
- Фильтрация стоп-слов

### Layer 2: **JVGR** (Jarvis VectorGraph)
- Процедурный граф команд
- Числовые идентификаторы действий
- Поддержка цепочек операций

### Layer 3: **Executor**
- Реальные действия с файловой системой
- Поиск (через hybrid_engine)
- Открытие/удаление/копирование файлов
- Вычисления (математические выражения)
- Запуск программ

### Layer 4: **MemoryGraph**
- Эпизодическая память
- Хранение результатов действий
- Индекс по объектам (O(1) поиск)
- Автоматическое удаление старых записей (FIFO)

### Layer 5: **ContextResolver**
- Использование прошлого опыта
- Автодополнение неполных команд
- "открой файл" → открывает последний файл

## 📈 ПРОИЗВОДИТЕЛЬНОСТЬ

| Команда | Время | Память |
|---------|-------|--------|
| Парсинг команды | 1-3 ms | < 1 MB |
| Поиск по объекту | O(1) | - |
| Сохранение в память | O(1) | 10-25 MB (10k записей) |
